DALL路E Image Generation

Overview

This project utilizes OpenAI's DALL路E-3 model to generate images based on textual descriptions. The script processes image data and displays AI-generated visuals.

Required Tools

  • Python 3.11+
  • OpenAI API
  • PIL (Pillow) for image handling
  • Base64 for encoding/decoding images
  • Dotenv for environment variable management

Step-by-Step Explanation

1. Import Necessary Libraries

import os
import json
from dotenv import load_dotenv
from openai import OpenAI
import base64
from io import BytesIO
from PIL import Image

These libraries help in environment management, API interaction, and image processing.

2. Load OpenAI API Key

load_dotenv(override=True)
openai_api_key = os.getenv('OPENAI_API_KEY')
if openai_api_key:
    print(f"OpenAI API Key exists and begins {openai_api_key[:8]}")
else:
    print("OpenAI API Key not set")

This ensures secure storage of the API key in a .env file.

3. Initialize OpenAI Client

openai = OpenAI()

The OpenAI client is initialized to interact with the DALL路E API.

4. Generate an Image

def artist(city):
    image_response = openai.images.generate(
        model="dall-e-3",
        prompt=f"An image representing a tradition in {city}, showing everything unique about {city}",
        size="1024x1024",
        n=1,
        response_format="b64_json",
    )
    image_base64 = image_response.data[0].b64_json
    image_data = base64.b64decode(image_base64)
    return Image.open(BytesIO(image_data))

This function generates an image based on a given city name.

5. Display the Generated Image

image = artist("Delhi")
display(image)

The generated image is displayed using PIL.
