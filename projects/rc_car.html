RC Car using QtPi (Low Code Platform)

An RC car is a Remote Controlled car, which can move in four directions. It could be controlled with an android device using Bluetooth.

Introduction

This project showcases how to build an RC car using hardware and develop a mobile app to control it using a low-code programming platform.

Components Used

  • QtPi Board
  • DC Motors (2)
  • Building Blocks
  • Code2platform to develop mobile app

Building Hardware Structure

Build the Structure using building blocks. Connect left motor to Port M1 and Right motor to port M2 on QtPi Motherboard

Mobile App Development using Low Code Platform

The mobile app is developed using the low-code platform Code2Play which provides a digital version of all physical components, such as motors in the case of the RC car.

Moving Forward

- When the 'Forward' button is pressed (TouchDown event), both motors rotate at same speed (Ex: 255) in the clockwise direction.

- When the 'Forward' button is released (TouchUp event), both motors stop.

Moving Backward

- When the 'Backward' button is pressed (TouchDown event), both motors rotate at same speed (Ex: 255) in the anti-clockwise direction.

- When the 'Backward' button is released (TouchUp event), both motors stop.

Moving Left

- One of the way to turn left is shown here

- When the 'Left' button is pressed (TouchDown event), left motor rotate in anti clockwise and right motor rotate in clockwise direction.

- When the 'Left' button is released (TouchUp event), both motors stop.

Moving Right

- When the 'Right' button is pressed (TouchDown event), left motor rotate in clockwise and right motor rotate in anti clockwise direction.

- When the 'Right' button is released (TouchUp event), both motors stop.
